# 视频转 GIF 工具 - 使用说明

## 简介

一个本地运行的视频转 GIF 工具，通过浏览器操作，支持参数调节和批量转换。

## 环境要求

| 依赖 | 说明 |
|------|------|
| **Python 3** | 运行后端服务 |
| **Flask** | Python Web 框架，安装命令：`pip3 install flask` |
| **ffmpeg** | 视频处理工具，安装命令：`brew install ffmpeg` |

## 快速启动

### 方式一：双击启动（推荐）

在 Finder 中双击 `start.command` 文件，终端会自动启动服务并打开浏览器。

> 首次运行时，macOS 可能提示"无法验证开发者"。请前往「系统设置 → 隐私与安全性」中点击"仍要打开"。

### 方式二：命令行启动

```bash
cd ~/Downloads/video2gif
python3 server.py
```

然后在浏览器中打开 http://127.0.0.1:5050

## 使用步骤

1. **上传视频** — 将视频文件拖拽到页面上传区域，或点击选择文件。支持一次上传多个文件（最多 10 个）。
2. **调整参数** — 上传后会显示参数设置面板，可调节以下选项：
   - **帧率 (FPS)**：1-30，值越高动画越流畅，文件越大。推荐 10-15。
   - **宽度**：输出 GIF 的宽度（像素），高度按比例自动计算。推荐 320-480。
   - **起始时间**：从视频的第几秒开始截取，默认 0（从头开始）。
   - **持续时长**：截取多少秒，默认 0 表示截取到视频结尾。
   - **画质模式**：
     - 高质量：使用 ffmpeg 双 pass 调色板技术，色彩更准确，速度较慢。
     - 快速：单次处理，速度更快但色彩可能有色带。
   - **循环次数**：0 表示无限循环，1 表示只播放一次。
3. **开始转换** — 点击「开始转换」按钮，所有文件会并行处理（最多同时 3 个），页面实时显示进度。
4. **下载结果** — 转换完成后，每个文件旁会出现「下载」和「预览」按钮。

## 支持的视频格式

MP4、AVI、MOV、MKV、WebM、FLV、WMV

## 文件大小限制

- 单个文件最大 500MB
- 单个文件转换超时时间为 10 分钟

## 目录结构

```
video2gif/
├── start.command      # 双击启动脚本
├── server.py          # 后端服务
├── 使用说明.md         # 本文件
├── templates/
│   └── index.html     # 前端页面
├── uploads/           # 上传的视频（临时）
└── output/            # 生成的 GIF（临时）
```

## 常见问题

**Q: 双击 start.command 没有反应？**
A: 右键点击文件 → 打开方式 → 终端。或在终端中直接运行 `bash start.command`。

**Q: 提示端口被占用？**
A: 说明上次的服务还在运行。在终端中执行 `lsof -i :5050` 查看进程，用 `kill <PID>` 结束它。

**Q: GIF 文件太大怎么办？**
A: 尝试减小宽度（如 320px）、降低帧率（如 8fps）、缩短持续时长。

**Q: 如何停止服务？**
A: 在运行服务的终端窗口中按 `Ctrl+C`。

## 清理文件

- 页面上的「清理服务器文件」按钮会删除 uploads/ 和 output/ 目录下的所有文件。
- 每次启动服务时，也会自动清理上次残留的临时文件。
